# Abacus Calculator

A simple and stylish calculator application for Android, built with Kotlin and Material Design components.

## Features

*   **Standard & Scientific Operations:** Performs basic arithmetic (add, subtract, multiply, divide) as well as scientific functions (square root, pi, power, percentage).
*   **Intuitive UI:** A clean, grid-based layout that's easy to use.
*   **Expression Editing:** Includes `AC` (All Clear) and `âŒ«` (Backspace) buttons for easy corrections.
*   **Parentheses Support:** Handles complex expressions with parentheses.

## Tech Stack

*   **Language:** Kotlin
*   **UI:** Android SDK with ViewBinding and Material Components.
*   **Layout:** `ConstraintLayout` and `GridLayout` for a responsive and organized UI.
*   **Calculation Engine:** `net.objecthunter:exp4j` for robust mathematical expression evaluation.

## Getting Started

1.  Clone this repository.
2.  Open the project in Android Studio.
3.  Build and run on an emulator or physical device.
